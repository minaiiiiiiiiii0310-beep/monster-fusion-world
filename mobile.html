📱 スマホだけで AIモンスター作成

PC を使わずに ボタンタップだけで モンスター画像を 生成 → ゲームに 反映できます。
1下の各モンスターで AI生成

緑 ImageFX（完全無料・推奨）または 青 Gemini・他のサービスのボタンをタップ。 プロンプトが自動コピー＆AIが新タブで開きます。AIの入力欄に 長押し→貼り付け。

注: 各サービスは Google/Microsoft 等のログインが必要です。

2気に入った1枚を保存（長押し→「画像を保存」）

4枚出るので 1枚を選び、写真アプリに保存。そのときファイル名は気にしなくてOK （あとで GitHub 側でリネームします）。

3GitHub にアップロード

下の 赤いボタンから GitHub の アップロード画面を直接開けます。 Microsoft アカウントではなく あなたの GitHub アカウントでログインしてください。

📤 アセットフォルダにアップロード

①「choose your files」または ドラッグエリアをタップ → ② 保存した画像を選択 → ③ ファイル名を sla1.png 等に変更（モバイル GitHub UI ではアップロード時にリネーム可能）→ ④ 下まで スクロールして「Commit changes」

41〜2分待ってゲームをリロード

GitHub Pages が自動で再デプロイされます。 ゲームを開く → リロードすると「🎨 AI画像を検出！」のトーストが出ます。

せってい → バトル表示 → 🎴 2Dカード を選ぶと、AI絵のカードでバトルできます。

💡 ヒント

  • ファイル拡張子は何でもOK: .png / .jpg / .jpeg / .webp どれでも自動認識
  • サイズも自動調整: 大きすぎる画像は CSS で自動縮小（多少 重くなる程度）
  • 背景透過したい場合: remove.bg でブラウザだけで処理可能
  • 正方形(1:1) 推奨: 縦長/横長でも表示はされるが、カードに余白が出る
  • iPhone でファイル名を変える: ファイルアプリで長押し → 名称変更
  • Android でファイル名を変える: ファイル → ⋮ → リネーム / または GitHub アップロード時に直接入力

📚 用語

  • 種族ID: sla1（スライムTier1）等。ファイル名は必ず この形式で
  • 有効な ID: 28 系統 × 5 ティア + 属性神8 + 巨神8 + オリジン = 157 種
  • 全 ID は HOW-TO.md に
